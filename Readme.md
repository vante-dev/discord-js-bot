# Features
- Commands Handler
- Slash Commands Handler
- Context Menu Commands Handler
- Events Handler
- Permission Handler
- Cooldowns

## Installation & Setup
```fix
Start.bat

you can find command & event example in ./src/Vante/Examples
Leave the ( ServerID ) in ( System.js ) file blank if you want to register slash commands globally.
```
